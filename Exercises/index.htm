E-R Exercises

Define entities and main relationships of the data needed to deal with the following situations (Note: use MySQL workbench to draw entities-relationship scheme):

  1. Set of inhibitor data, including nature of the inhibitor compound, enzyme inhibited,  source organism, type of inhibition (reversible, irreversible, etc.), inhibition constant, and literature reference. (Solution:: mwb png)
  2. Protein-Protein interactions data. Proteins involved (include references to Uniprot and/or PDB), interaction details (permanent, transient, etc), experimental data and literature references. (Solution:: mwb png)
  3. Genotype data, about 250000 SNPs for a large number of individuals. Consider two options: a) We should get information for individual SNPs, and b) only complete haplotypes should be retrieved. (Solution::png)
  4. User management, including specific levels of privilege like administration, readwrite data, read-only, and personal preferences. The model should be open enough to be used in any application. (Solution:: mwb png)
  5. Expression analysis for a set of genes, and experimental conditions. For the stored genes should allow including annotations and references to other DBs. (Solution:: mwb png)
  6. A metabolic network, including, enzymes, metabolites and chemical processes involved. (Solution:: mwb png)

Evaluated exercises (by 12th Feb)

You are the data manager of a bioinformatics support service and need to build a database to store data from your users' studies. The following cases have just arrived. Define a data model (entities, atrributes and relationships) to hold data from ONE of them.

  • A series of GWAS (Genome Wide Association Studies) based on SNP-arrays data. Data includes: 1) a variable set of phenotipic traits. 2) Reference, supplier and genes included in SNP-array used in the study, 3) Sample and user details, 4) GWAS results: genes and associated statistics.
  • A series of proteomics analysis. Data includes: 1) Sample and user details, 2) Experimental protocols used, 3) Protein sequences found, 4) Identified proteins from sequences
